GeoIP database update in multiple webserver environment using Windows PowerShell scripting
http://blog.mathiaskunto.com/2012/01/15/geoip-database-update-in-multiple-webserver-environment-using-windows-powershell-scripting/

Important: This script uses gzip.exe to unpack the GeoIP database. It may be downloaded from the gzip homepage: http://www.gzip.org/

Important: It also requires the Get-WebFile (v3.6) by Joel Bennett for retrieving the database file. It may be downloaded from the PowerShell Code Repository: http://poshcode.org/417
